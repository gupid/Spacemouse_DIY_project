# Spacemouse DIY Project

这是一个开源的 DIY 3D 鼠标 (Spacemouse) 工程。它基于沁恒(WCH)的 CH585 系列微控制器，旨在为 3D 设计、建模和工程人员提供一个高性价比的 3D 控制器解决方案。该设备集成了 3D 鼠标、普通鼠标和键盘功能，并支持有线、蓝牙(BLE)和私有 2.4G RF 连接。

## 硬件概述

主控为CH585，接收器为CH582F，键盘扫描芯片为CH450，均为沁恒产品。陀螺仪使用MPU9250。

## 主要功能

* **六轴自由度控制**: 使用 MPU9250 传感器，实现平移、旋转、缩放等直观的 3D 空间操作。
* **多模式切换**: 支持USB、BLE、2.4GHz三种模式连接
* **可编程按键**: 可在USB/2.4GHz模式下通过上位机自由修改。

## 工程文件结构

本仓库主要包含两个核心部分的固件：

* **`Spacemouse_develop/`**: 这是 3D 鼠标设备本身的主体固件。
  * `APP/`: 应用层代码，包括主逻辑 `main.c` 和模式管理 `mode_manager.c` 等。
  * `Driver/`: 驱动程序，用于控制 MPU9250 传感器、CH450 按键芯片和 WS2812B RGB LED。
  * `Middleware/`: 中间件代码，处理运动数据、HID 报告和无线通信。
  * `HAL/`, `LIB/`, `StdPeriphDriver/`: 硬件抽象层、库文件和沁恒官方提供的标准外设驱动。
* **`spacemouse_RF_To_USB/`**: 这是CH585版本的 2.4G RF 无线接收器（USB Dongle）的固件。
  * `src/`: 接收器的主逻辑，负责接收 2.4G 信号并将其转换为 USB HID 数据。
  * `HAL/`, `LIB/`, `StdPeriphDriver/`: 与设备端类似的底层驱动和库。
* **`spacemouse_RF_To_USB_CH582/`**: 这是CH582版本的 2.4G RF 无线接收器（USB Dongle）的固件。
* `上位机`：这是和设备配套使用的，可以通过此HTML文件自由配置键值和转轴灵敏度。

## 写在最后

此工程为2025年嵌入式芯片与系统设计竞赛应用赛道——沁恒赛道的作品。实际上，到本次大赛结束时，我们的作品依然没有做完。这份决赛的代码，目前实现的功能有：USB模式、2.4GHz模式、上位机配置、比较稳定的陀螺仪控制。而仍然有待提升的有BLE模式，BLE模式极其不稳定，键盘和摇杆都有很大问题。

同时主板PCB设计也存在一定问题，锂电池的充电功能可行，但供电时却电压不足。

这个作品仍未走到它的结局，而我们却已然失去了继续优化它的动机。这是由三个大二学生历时两个月做出来的作品，这一路磕磕绊绊，但最终还是把它送上了国赛的舞台。它凝结了几十个日夜的心血，是我们无比宝贵的作品。

如果有缘，或许未来还会继续优化，我能感到，自己心中还有很多idea在熊熊燃烧。也欢迎所有朋友，如果你也对此感兴趣，希望这份工程能帮到你。
